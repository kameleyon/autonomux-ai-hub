# Autonomux — Your AI Workforce, One Click Away

An AI Agents Marketplace where users browse, deploy, and manage autonomous AI agents to automate tasks.

## Tech Stack

- **Frontend:** React 18 + TypeScript + Vite
- **UI:** shadcn/ui + Tailwind CSS
- **Backend:** Supabase (Auth, Database, Edge Functions, Realtime)
- **Payments:** Stripe Checkout
- **AI:** OpenRouter (Claude Sonnet)

## Development

```sh
npm install
npm run dev
```

## Environment Variables

Copy `.env.example` to `.env` and fill in your values:

- `VITE_SUPABASE_URL` — Supabase project URL
- `VITE_SUPABASE_PUBLISHABLE_KEY` — Supabase anon key
